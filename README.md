## Purpose

There are a number of R cookbooks and also books about network analysis and
network science. Also, there are a lot of great tutorials and blogs with
information on how to do network analysis using tidygraph and ggraph. But there
is no cookbook for using tidygraph and ggraph to do common network analysis
work. So this book here aims to fill that gap.
